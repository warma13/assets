## 第七章设计方案：「瘴毒密林」

### 主题定位

前六章分别是 荒原（灰黄）→ 冰封（冰蓝）→ 熔岩（熔红）→ 墓域（幽紫）→ 深海（深蓝）→ 雷漠（琥珀金/电紫），元素覆盖了火、冰、毒、水、物理、奥术。第七章走**瘴毒密林**路线，理由：

* **翠绿/猩红**色调与前六章形成鲜明视觉反差——原始丛林的墨绿配合毒液的猩红荧光，是全新的视觉组合
* **毒+物理**双元素主导：毒在第四章后沉寂了两章，物理在第六章只作为坦克属性出现，第七章让两者以"瘴毒密林"形态回归
* 热带雨林毒物（瘟虫、毒蛇、黑豹、绞藤、孢子虫、毒蜂、铁木树人、沼地巫师）有丰富的视觉表现力
* 可引入新机制：**毒蛊叠加（venomStack）**（叠加式毒素DoT，每层按最大HP百分比持续掉血）、**孢子云（sporeCloud）**（怪物死亡释放减攻速孢子云）

#### 新机制说明

| 机制 | 字段 | 作用 | 实现要点 |
|------|------|------|----------|
| **毒蛊叠加（venomStack）** | `venomStack = { dmgPctPerStack, stackMax, duration }` | 每次攻击叠毒蛊，每层每秒按最大HP%持续掉血 | 在 `EnemyAttackPlayer` 中叠层；在 `UpdateDebuffs` 中按层数扣血 |
| **孢子云（sporeCloud）** | `sporeCloud = { atkSpeedReducePct, duration }` | 怪物死亡释放孢子云，降低玩家攻击速度 | 在 `OnEnemyDeath` 中检查并施加 debuff |

**韧性交互**：所有新 debuff 都需要经过韧性抗性计算：
```lua
local resist = GameState.GetDebuffResist()
local actualReducePct = rawReducePct * (1 - resist)
local actualDuration = duration * (1 - resist * 0.5)
```

#### 关键规律

1. **火元素最优** — 所有 8 种普通怪都弱火（-15%~-30%），两个 Boss 也弱火，火系是全章通用最优解
2. **水元素次选** — 蝮蛇(-15%)、黄蜂(-10%)、树人(-15%)、阿拉克涅(-15%)弱水
3. **冰元素有限** — 仅孢子潜伏者(-15%)和耶梦加得(-10%)弱冰
4. **毒元素被高抗** — 甲虫(30%)、蝮蛇(40%)、巫师(40%)、阿拉克涅(50%)高毒抗，毒系在本章极为吃力
5. **物理被高抗** — 黑豹(30%)、藤蔓(40%)、树人(50%)、耶梦加得(40%)高物理抗
6. **Boss 弱点统一** — 两个 Boss 都弱火，火是唯一通用元素

**策略总结**：本章主推**火元素**武器。上一章（雷鸣荒漠）的风暴裁决套装虽以奥术为主，但本章新套装「毒牙之焰」专门提供火伤加成和点燃能力——这是精心设计的套装迭代。

### 套装设计

#### 套装一：「毒牙之焰」— 攻击型（火焰点燃 + 焰爆处决）

**核心幻想**：以火焰净化毒林，将毒蛊转化为爆发火力。对毒系怪物拥有特化的点燃优势。

**ID**: `venom_fang`
**主题色**: `{200, 80, 30}` (烈焰橙红)
**所属章节**: 7

**2件效果：毒火之焰**
> 火焰伤害 +25%。攻击命中时 20% 概率点燃目标（对毒系怪物提升至 35%），每秒造成 3%ATK 火焰伤害，持续 4 秒。

```lua
bonuses = {
    [2] = {
        desc = "火伤+25%。攻击20%几率点燃(毒系35%),3%ATK/秒,4秒",
        stats = { fireDmg = 0.25 },
        buff = {
            id = "venom_fang_2", trigger = "onHit",
            igniteChance = 0.20, igniteChanceVsPoison = 0.35,
            igniteDmgPct = 0.03, igniteDur = 4.0,
        },
    },
}
```

* 火伤 +25% 是直接战力提升，过渡阶段就有感知
* 20%基础点燃率 → 对毒系怪物 35%，本章大部分怪都是毒系，触发率极高
* 配合全章火元素弱点，形成"火克毒"的清晰策略

**4件效果：焰爆净化**
> 对燃烧中的目标连续命中 3 次后触发「焰爆」：造成 **250%ATK** 的火焰伤害，并清除目标身上所有毒蛊层数，同时获得 10% 攻速提升，持续 4 秒。每个目标每 5 秒最多触发一次。

```lua
[4] = {
    desc = "燃烧目标命中3次→焰爆250%ATK+清毒蛊+攻速10%4秒,CD5秒/目标",
    buff = {
        id = "venom_fang_4", trigger = "onHitBurning",
        hitsRequired = 3, blastDmgMul = 2.50, blastElement = "fire",
        clearVenomStacks = true, atkSpeedBonus = 0.10, atkSpeedDur = 4.0,
        cd = 5.0,
    },
},
```

| 要素 | 具体设计 | 作用 |
|------|---------|------|
| **焰爆伤害** | 250%ATK 火焰伤害 | 高额爆发，配合火弱点效果极佳 |
| **清除毒蛊** | 焰爆触发时清除目标毒蛊层数 | 直接对抗本章核心威胁机制 |
| **攻速提升** | 10% 攻速 4 秒 | 连续焰爆循环加速 |
| **CD 限制** | 每目标 5 秒一次 | 控制 DPS 上限 |

**与本章怪物的联动**：
- 荆棘蝮蛇(火抗-30%)+毒蛊叠加 → 焰爆精准克制：火伤高效+清除毒蛊
- 铁木树人(火抗-30%, 500HP, 45DEF) → 焰爆250%ATK无视高防，配合火弱点秒伤极高
- 毒雾黄蜂蜂群+毒蛊 → 点燃率对毒系35%，快速点燃→焰爆清理

#### 套装二：「丛林吐息」— 防御型（毒免+毒蛊转化+孢子利用）

**核心幻想**：与密林共生，将敌人的毒素化为己用。毒蛊层数不再是威胁，而是回复来源。

**ID**: `grove_breath`
**主题色**: `{50, 160, 80}` (翠绿)
**所属章节**: 7

**2件效果：共生吐息**
> 毒素抗性 +35%，防御力 +18%，韧性 +12。毒蛊层数转化为每层 0.5% 最大HP/秒的回复效果（变害为利）。

```lua
bonuses = {
    [2] = {
        desc = "毒抗+35%,防御+18%,韧性+12。毒蛊层数转化为每层0.5%HP/秒回复",
        resist = { poison = 0.35 },
        statsMul = { def = 0.18 },
        stats = { ten = 12 },
        buff = {
            id = "grove_breath_2", trigger = "passive",
            venomStackHealPct = 0.005,  -- 每层每秒回复0.5%最大HP
        },
    },
}
```

* 毒抗 +35% 大幅削减本章大量毒系怪物的伤害
* DEF +18% + 韧性 +12（=+24%抗性）双重防御层
* **核心卖点**：毒蛊层数反转为回复——蝮蛇叠的毒蛊越多，玩家回血越快
* 与磐岩之盾不同：磐岩是物理防御，丛林吐息是毒系防御+转化

**4件效果：密林觉醒 + 丛林不灭**
> ① 击杀带孢子云的怪物时，获得 15% 攻速提升，持续 5 秒。
> ② 受到致命伤害时**免死一次**（每关重置），触发后获得 3 秒无敌 + 立即恢复 25% 最大 HP，并释放 150%ATK 毒素脉冲（全屏毒伤）。

```lua
[4] = {
    desc = "击杀孢子怪+15%攻速5秒;致命保护(每关1次)+3秒无敌+回25%HP+150%ATK毒脉冲",
    buff = {
        id = "grove_breath_4", trigger = "onKillSpore",
        atkSpeedBonus = 0.15, atkSpeedDur = 5.0,
        fatalProtect = true, fatalInvulDur = 3.0,
        fatalHealPct = 0.25, fatalPulseDmgMul = 1.50,
        fatalPulseElement = "poison", perStage = true,
    },
},
```

| 要素 | 具体设计 | 作用 |
|------|---------|------|
| **击杀孢子怪加速** | 击杀 sporeCloud 怪物 +15% 攻速 | 变孢子云从威胁为增益来源 |
| **免死保底** | 致命伤免死1次/关 | 延续历代防御套传统 |
| **3秒无敌** | 免死后3秒无敌 | 充足的喘息空间 |
| **25%HP回复** | 免死后立即回血 | 比磐岩之盾（20%）更高 |
| **毒素脉冲** | 150%ATK 全屏毒伤 | 反击清场能力 |

**与本章怪物的联动**：
- 荆棘蝮蛇+毒蜂群叠毒蛊 → 2件将毒蛊变回复，完全对抗核心威胁
- 孢子潜伏者/铁木树人死亡释放孢子云 → 4件变孢子云为攻速buff
- Boss 弹幕+吐息高爆发 → 免死+无敌+25%回血保障生存

#### 两套套装策略对比

| 维度 | 毒牙之焰 | 丛林吐息 |
|------|----------|----------|
| **定位** | 主动火焰输出 | 防御+毒免+转化 |
| **触发方式** | 我打敌人 → 点燃 → 焰爆 | 被叠毒蛊 → 转回血；被打死 → 免死反击 |
| **核心克制** | 毒系怪物（点燃率35%）、高防坦克（焰爆250%ATK） | 毒蛊叠加（转回血）、孢子云（转攻速） |
| **爆发元素** | 火（焰爆） | 毒（脉冲） |
| **生存能力** | 无 | 毒抗+DEF+韧性+毒蛊回复+免死+无敌 |
| **适合 Boss** | 阿拉克涅（毒弹幕需要火力压制）、耶梦加得（火弱点） | 阿拉克涅（毒蛊回复对冲弹幕伤害） |
