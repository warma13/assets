## 设计理念

### 章节定位

| 维度 | 说明 |
|------|------|
| 主元素 | fire（火） |
| 辅助元素 | physical（物理）、arcane（奥术） |
| 抗性系数 | 1.2（与第13、14章配对） |
| 色调 | {240, 80, 30}（炽焰橙红） |

fire 在第 13 章（冰）和第 14 章（毒）中一直是玩家的最优反制元素——两章 Boss 都以 fire 为最大弱点。第 15 章将火从"玩家的武器"反转为"敌人的武器"，迫使长期依赖火系 Build 的玩家重新构建阵容，形成元素博弈的戏剧性转折。

### 元素反转设计

| 维度 | 第13章（冰）/ 第14章（毒） | 第15章（火） |
|------|--------------------------|------------|
| 玩家最优元素 | fire（融化/焚净） | **water**（淬灭） |
| Boss 最大弱点 | fire (-0.25~-0.30) | **water** (-0.30) |
| 护盾弱点反应 | melt / purify（火打） | **quench**（水打） |
| 设计意图 | 火队横扫 | 火队失效，必须转水 |

### 与第 13 章（冰）和第 14 章（毒）的三章对比

| 维度 | 第13章 冰 | 第14章 毒 | 第15章 火 |
|------|----------|----------|----------|
| 核心威胁 | 空间压缩 + 移速衰减 | 防御腐蚀 + DoT 叠加 | DPS 剥夺 + 受伤增幅 |
| 压制方式 | 冻住你、困住你、让你跑不掉 | 毒蚀你、腐朽你、让你越来越脆 | 灼烧你、耗竭你、让你打不动 |
| 致死模式 | 被限制在小范围内吃满伤害 | DoT 和腐蚀叠加到阈值后雪崩式崩溃 | 攻速归零+受伤增幅，无法输出被活活烧死 |
| Build 偏好 | 高移速 + 冰抗 | 高韧性 + 毒抗 + 净化/回复 | 高爆发 DPS + 火抗 + 水系元素 |
| 反制元素 | fire（融化反应） | fire（焚净反应） | **water（淬灭反应）** |
| decay 目标 | moveSpeed（跑不动） | def（扛不住） | **atkSpeed（打不动）** |
| 战斗节奏 | 走位挑战 | 生存挑战 | **DPS 竞速挑战** |

### 两个 Boss 的定位差异

| 维度 | 伊格尼斯（中 Boss） | 萨拉曼德（终 Boss） |
|------|-------------------|-------------------|
| 战斗风格 | 烈焰倾泻型：高频弹幕叠灼烧 + 火元素蜂涌 | 焦土碾压型：攻速剥夺 + 受伤增幅 + 焚天终焉 |
| AI 决策重点 | 控制灼烧层数 + 清火精灵 + 打焰核削弱 Boss 攻击 | 维持攻速不被耗竭 + 应对淬灭护盾 + 阶段三焰核抢时 |
| Build 偏好 | 高移速 + AOE → 快速清火精灵减少灼烧源 | 高爆发 + 水系 + 火抗 → 在攻速衰减前击杀 |
| 失败原因 | 灼烧叠满 + 火精灵全场放火，持续 DoT 致死 | 攻速被削到极低 + 受伤增幅叠满，DPS 崩盘被烧死 |
| 阶段数 | 2 阶段 | 3 阶段 |

### 弱点

| 元素 | 抗性 | 说明 |
|------|------|------|

| ice | -0.20 | 次要弱点（冷却降温） |

| poison | 0.30 | 高抗（火焰焚毁毒素） |
| fire | 0.50 | 极高抗 |

### 阶段一（100%→55%）：烈焰洗礼

Boss 以高频弹幕铺满战场火力，同时召唤火精灵形成数量压制。每颗弹命中都叠加"灼烧"，灼烧层数越高每秒火伤越痛且攻速越慢，迫使玩家要么精准走位控制灼烧层、要么在灼烧叠满前尽快击杀。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_barrage** | count: 20, spread: 150°, dmgMul: 0.75, speed: 200, interval: 5.0s |
| 地面威胁 | **ATK_spikes** | count: 5, radius: 32, delay: 1.0s, dmgMul: 1.1, lingerTime: 6s, interval: 7s |
| 召唤 | **SUM_minion** | monsterId: flame_imp, count: 5, interval: 8s |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| barrage.onHit | 每颗弹命中叠加 1 层"灼烧"（每层 1.8% ATK/秒 火伤 DoT + 攻速降低 3%，持续 5s，最多 8 层）。新命中刷新全部层的持续时间 |
| spikes.linger | 残留熔岩池，站在上面每 0.5s 叠加 1 层灼烧 + 0.2×ATK 火伤。熔岩池是持续的叠烧源 |
| minion.onDeath | 火精灵死亡时爆炸留下火池（半径 20，持续 3s），范围内叠加 1 层灼烧 |

**灼烧机制详解**:
```
灼烧 (Blaze):
- 每层: 1.8% ATK/秒 火伤 DoT + 攻速 -3%
- 最大层数: 8 层 (= 14.4% ATK/秒 = 828×0.144 = 119.2 DPS + 攻速 -24%)
- 持续时间: 5s (新命中刷新全部)
- 叠加源: 弹幕命中 / 熔岩池接触 / 火精灵死亡爆炸
- 净化: 3s 不受新叠加，每秒自然消退 1 层
- 与冰章 frostbite/毒章蚀毒的区别: 同时兼具 DoT 和属性削弱（攻速），是两者的混合威胁
```

**AI 威胁信号**:
- 弹幕：每颗弹体注册 dangerZone（半径 10），AI 可横向移动躲避减少灼烧叠加
- 熔岩池：预警阶段注册 dangerZone（半径 32，priority 0.85），优先级高于冰柱——因为站在熔岩池里会持续叠灼烧且掉血
- 灼烧层数：注册 debuffStack（currentStacks），AI 根据层数动态调整：低层(1-3)可激进输出，高层(5+)优先走位躲避

### 阶段二（55%→0%）：焰翼领域

弹幕从扇形变为环形（全方向），新增领域和焰核图腾。焰核存活时增强 Boss 攻击力（ATK +20%），使弹幕和灼烧 DoT 都更痛。AI 面临三重决策：躲弹幕控灼烧层、脱离领域、打焰核削弱 Boss 输出。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_barrage** | count: 16, spread: 360°（全向但更稀疏）, dmgMul: 0.75, speed: 170, interval: 6s |
| 领域 | **CTL_field** | radius: 120, dmgMul: 0.30, tickRate: 0.5s, duration: 8s, cd: 13s, hpThreshold: 0.55 |
| 减伤 | **DEF_armor** | hpThreshold: 0.40, dmgReduce: 0.55, duration: 5s, cd: 12s |
| 回血源 | **DEF_crystal** | count: 2, hpPct: 0.02, healPct: 0.015, spawnInterval: 11s, spawnRadius: 85 |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| field.effect | 领域内每 0.5s 叠加 1 层灼烧（与弹幕独立叠加），同时回复压制 40%（领域内回血效果降低） |
| crystal.aura | 焰核存活时：Boss ATK +20%（弹幕伤害和灼烧 DoT 基数同步提升）。焰核 = Boss 输出倍率放大器 |
| crystal.onDestroy | 焰核摧毁时爆裂释放冷却波，清除玩家 3 层灼烧（打掉焰核有正面收益） |
| armor.visual | 岩浆硬化甲壳，暗红色熔岩护甲包裹效果 |

**AI 威胁信号**:
- 领域：dangerZone（半径 120，持续 8s）。领域内不仅掉血还叠灼烧+减回复，脱离优先级高
- 焰核：priorityTarget（priority 0.75）。打掉焰核三重收益——中断 Boss 回血 + 削弱 Boss ATK + 清除自身 3 层灼烧
- 护甲期间：bossState（dmgReduce 0.55）。可趁此去清火精灵/打焰核

**伊格尼斯战斗节奏**:
```
阶段一: 左右移动躲弹幕(150°扇形) + 走出熔岩池 + 清火精灵(注意死亡火池)
        核心: 控制灼烧层数在 3 层以下，保持攻速健康
    ↓ HP 55% 转换演出 (全场+2层灼烧)
阶段二: 躲弹幕(360°更稀疏) + 脱离领域 + 优先打焰核(削弱Boss ATK+清灼烧层)
        核心: 焰核存活时 Boss ATK×1.2, 灼烧 DoT 更痛, 必须在灼烧叠满前摧毁
```

## Boss 2：焚天魔君·萨拉曼德 (boss_inferno_sovereign)

| 属性 | 值 |
|------|-----|
| HP | 5,640,000 |

| DEF | 156 |
| 速度 | 12 |

| element | fire |
| isBoss | true |

### 弱点

| 元素 | 抗性 | 说明 |
|------|------|------|
| water | -0.25 | 最大弱点（淬灭烈焰） |
| ice | -0.15 | 次要弱点（冷却抑制） |
| arcane | -0.10 | 微弱弱点 |
| physical | 0.10 | 微抗 |
| poison | 0.35 | 高抗（火焰焚毁毒素） |
| fire | 0.60 | 极高抗 |

### 阶段一（100%→60%）：灼热试探

Boss 以宽角度火焰吐息和熔岩喷泉建立空间控制，同时召唤灼热守卫分散火力。吐息命中叠加"焚灼"debuff（增加受到的伤害），与阶段二的 CTL_decay 形成"打不动+扛不住"的双重压制链。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_breath** | angle: 75°, range: 165, dmgMul: 0.50, tickRate: 0.3s, duration: 2.0s, interval: 7s |
| 地面威胁 | **ATK_spikes** | count: 4, radius: 35, delay: 1.0s, dmgMul: 1.2, lingerTime: 6s, interval: 8s |
| 嘲讽守卫 | **SUM_guard** | count: 2, hpPct: 0.014, atkMul: 0.40, tauntWeight: 0.55, interval: 13s |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| breath.onHit | 每次命中叠加"焚灼"1 层（每层增加受到的所有伤害 3%，最多 10 层 = +30% 受伤增幅，持续 8s，新命中刷新）。为阶段二 CTL_decay 做铺垫 |
| spikes.linger | 残留熔岩池，站在上面每秒受 0.35×ATK 火伤 + 回复压制 40%（熔岩池内回血效率大幅降低） |
| guard.aura | 守卫自带半径 50 的灼热光环（光环内每秒叠 1 层焚灼），不清守卫则焚灼快速叠满 |

**焚灼机制详解**:
```
焚灼 (Scorch):
- 每层: 受到的所有伤害 +3%（含普攻、技能、DoT、环境伤害）
- 最大层数: 10 层 (= 受伤 +30%)
- 持续时间: 8s (新命中刷新全部)
- 叠加源: 吐息命中 / 守卫光环 / 阶段二 decay 被命中时额外叠加
- 效果: 受伤增幅 → 所有伤害来源都更痛 → 与 DoT/领域形成正反馈
- 与毒章"腐蚀"的区别: 腐蚀降 DEF(减法)，焚灼增受伤(乘法)。焚灼在低 DEF 时效果更致命
```

**AI 威胁信号**:
- 吐息：dangerZone（扇形 75°，半径 165，持续 2.0s）。AI 可向侧面移动躲出，但持续时间比前两章更长
- 熔岩池：dangerZone（半径 35，priority 0.8）。熔岩池的回复压制使得"硬站熔岩靠吸血扛"策略失效
- 守卫：taunt（weight 0.55）+ dangerZone（半径 50 光环）。守卫不仅嘲讽还自带焚灼光环，必须优先清理

### 阶段二（60%→30%）：焦土碾压

核心机制转变：CTL_decay 持续自动削减攻速，护盾考验元素构建（首次要求 water 而非 fire），火墙缩小安全区。与前两章阶段二对比——冰章压缩空间+减速，毒章剥离防御+压制回复，火章剥夺 DPS+增幅受伤。三章形成"跑不动→扛不住→打不动"的递进关系。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_breath** | angle: 105°（扩大）, range: 165, dmgMul: 0.60, tickRate: 0.3s, duration: 2.0s, interval: 7s |
| 衰减 | **CTL_decay** | hpThreshold: 0.60, stat: atkSpeed, reducePerSec: 0.018, maxReduce: 0.35, bonusOnHit: 0.04 |
| 障壁 | **CTL_barrier** | count: 2, duration: 7s, contactDmgMul: 0.40, interval: 12s |
| 领域 | **CTL_field** | radius: 135, dmgMul: 0.35, tickRate: 0.5s, duration: 9s, cd: 15s, hpThreshold: 0.60 |
| 反应护盾 | **DEF_shield** | hpPct: 0.04, bossDmgReduce: 0.80, duration: 10s, cd: 20s, hpThreshold: 0.50, baseResist: 0.50 |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| decay.stat | **atkSpeed**（与冰章的 moveSpeed、毒章的 def 不同——火章削攻速）。被吐息/熔岩池命中时额外叠 4% |
| decay.combo | 攻速衰减与焚灼独立计算但效果协同：decay -35% atkSpeed（DPS 大幅下降）+ 焚灼 10 层 +30% 受伤（扛不住）= 既打不死 Boss 又被 Boss 打死。实际受攻速下限保护（最低保留 15% 基础攻速） |
| barrier.onContact | 碰触火墙：灼烧 2.5s（每秒 0.45×ATK 火伤）+ 叠加 3 层焚灼。火墙比冰墙和毒壁的惩罚更直接暴力 |
| field.effect | 领域内回复压制 55% + 每 0.5s 叠 1 层焚灼。与 decay 叠加形成加速恶化 |
| shield.reaction | 见下方详细定义 |

**反应护盾特调（淬灭反应）**:

```
shield_reaction: {
    weakReaction: "quench"           -- 水打火 = 淬灭
    weakElement: "water"
    weakMultiplier: 2.8              -- 水元素对护盾 ×2.8 伤害

    wrongHitEffects: {
        fire: {                      -- 火打火盾：最重惩罚
            shieldHeal: 0.10         -- 护盾回复 10% 自身 HP
            bossHeal: 0.03           -- Boss 回复 3% maxHP
        }
        poison: {                    -- 毒打火盾：毒素被焚烧反噬
            dmgFactor: 0.5           -- 伤害 5 折
            dotOnSelf: 0.015         -- 每秒自伤 1.5% maxHP（3s）
        }
        ice: {                       -- 冰打火盾：冰融蒸发，效率低
            dmgFactor: 0.7           -- 伤害 7 折
            scorchSelf: 2            -- 自身叠 2 层焚灼
        }
        physical: {                  -- 物理打火盾：灼热反弹
            reflect: 0.25            -- 反弹 25% 伤害给玩家
            atkSpeedReduce: 0.10     -- 攻速 -10%（3s）
        }
        arcane: {                    -- 奥术打火盾：中性偏弱
            dmgFactor: 0.60          -- 伤害 6 折，无额外惩罚
        }
    }

    timeoutPenalty: {                -- 10s 内未击破
        type: "explode"
        dmgMul: 1.5                  -- 护盾爆炸对全场造成 1.5×ATK 火伤
        scorchStacks: 5              -- 全场叠加 5 层焚灼
    }
}
```

**AI 威胁信号**:
- 领域：dangerZone（半径 135）。领域内回复压制 + 叠焚灼，必须脱离
- 火墙：dangerZone（矩形）。碰触惩罚极其暴力（持续灼烧 + 叠焚灼），AI 将火墙纳入移动禁区
- 护盾：priorityTarget（priority 0.8）+ bossState。AI 切换目标打护盾，**水队优势**（首次不再是火队优势）
- 衰减：debuff。AI 感知攻速持续下降，判断 DPS 窗口剩余时间，可能触发激进输出策略
- 焚灼层数：debuffStack。AI 根据当前受伤增幅判断风险等级

### 阶段三（30%→0%）：焚天终焉

全面碾压阶段。漩涡将你拉入灼热核心，被 decay 叠满的攻速衰减使你几乎无法输出，而焚灼增幅让每一点伤害都被放大。限时焰核迫使 AI 在"打焰核保命"和"抢 Boss 最后血量"间做抉择。与前两章阶段三对比——冰章用移速衰减困住你，毒章用防御衰减让你脆如纸，火章用攻速衰减让你无力还击。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 减伤 | **DEF_armor** | hpThreshold: 0.30, dmgReduce: 0.68, duration: 6s, cd: 14s |
