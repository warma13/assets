### 主题定位

### 怪物机制设计思路

 **吸血（新机制）** ：诅咒骑士攻击时回复自身HP。实现上只需在怪物攻击命中后加一行 `hp = hp + dmg * lifestealRate`，不需要改底层系统。这个机制让玩家必须追求高DPS——如果输出不够，怪物会越打越难死。

 **治疗光环（亡灵侍僧）** ：每8秒给范围内友军回5%最大HP。这迫使玩家优先集火侍僧，否则其他怪物的有效HP大幅提升。配合骷髅战士的高防和诅咒骑士的吸血，形成"不打侍僧就打不死"的策略深度。

## 第四章专属套装设计

### 设计原则

前三章的套装模式是 **一攻一守** ：

* 第一章：灰烬之焰（元素反应流） + 荒原守望（生存流）
* 第二章：碎冰者（火克冰输出） + 永冻之心（冰抗生存）
* 第三章：烈焰之心（攻速爆发） + 熔炉铸造（免死保命）

第四章应该延续这个模式，但 **套装效果要针对第四章怪物的核心机制** ，让玩家感受到"穿对套装，难度骤降"。

### 套装搭配策略

玩家在第四章的装备路线大致是：

| 阶段        | 推荐套装                        | 状态                  |
| ----------- | ------------------------------- | --------------------- |
| 1-3关       | 第三章旧装 + 零散第四章散件     | 还能打                |
| 4关（墙）   | 亡魂裁决 2件 或 不朽壁垒 2件    | 需要2件套效果才能扛住 |
| 5关 Boss    | 至少一套2件触发                 | 有奥术增伤/抗性才好打 |
| 6-7关       | 亡魂裁决 2 + 不朽壁垒 2（混搭） | 攻守兼备              |
| 8-9关（墙） | 任意一套凑满4件                 | 4件效果是质变         |
| 10关 Boss   | 亡魂裁决 4件（推荐）            | 反治疗克制Boss回血    |

这形成了清晰的装备驱动进度： **不是数值不够，而是机制上被克制** 。没有反治疗打不死吸血怪，没有奥术抗性扛不住爆发——这让"刷装备"变得有目标感，而不是纯粹堆数值。

## 抗性设计原则

### 基于亡灵主题的经典逻辑

* **毒免疫/高抗** ：亡灵没有活体组织，毒素无效——这是最核心的设定
* **火焰弱点** ：焚烧尸骨是经典克制手段——火成为本章优势元素
* **物理抗性** （部分）：幽灵和暗影形态的怪打不到实体
* **冰抗性** （部分）：已经是死物，冻结意义不大
* **奥术中立偏抗** ：部分亡灵本身就是奥术造物，对奥术有一定抵抗
* **水元素中立** ：无特殊交互

### 抗性数值含义

用百分比表示，正值=抗性（减伤），负值=弱点（增伤）：

* **50%抗性** = 伤害减半，基本打不动
* **30%抗性** = 明显肉，不推荐用这个元素
* **0%** = 正常伤害
* **-20%弱点** = 伤害提升20%，推荐用这个元素
* **-30%弱点** = 伤害提升30%，最优选择

终极 Boss 故意没有火焰弱点，甚至有轻微火抗。这意味着：

* 玩家用火武器一路顺风打到第10关，突然发现火不好使了
* 水元素（0%抗性）反而变成最优选——迫使玩家刷一把水元素武器备用
* 或者依赖套装的4件套效果（反治疗）来弥补元素伤害的不足
* 这创造了"最终Boss需要专门准备"的体验

## 套装设计

### 套装一：「净魂烈焰」— 攻击型（火焰积蓄爆发流）

 **核心幻想** ：不断攻击积蓄火焰能量，到达临界点后释放一次猛烈的净魂爆炸，焚尽周围一切亡灵。

**2 件效果：灼烧**

> 火焰增伤 +25%。攻击命中时 15% 概率**点燃**目标，使其在 3 秒内每秒受到 10% ATK 的火焰持续伤害。

* 不只是加一个数字，而是加了一个**可见的 DoT 效果**
* 点燃的敌人身上可以显示火焰特效，让玩家直观感受到套装在生效
* 配合火焰弱点怪物（骷髅-25%、白骨巨人-30%），灼烧的持续伤害也会被弱点放大
* 实现简单：每次命中 roll 15%，命中后给目标加一个 3 秒的 DoT 标记

**4 件效果：净魂爆燃**

> 每累计触发 **10 次点燃**后，释放「净魂爆燃」：以自身为中心产生火焰冲击波，对范围内所有敌人造成  **200% ATK 的火焰爆炸伤害** ，并使被命中的目标 **治疗和吸血效果降低 70%** ，持续 5 秒。内置冷却 6 秒。对**燃烧中**的目标额外造成 30% 伤害。

逐层拆解这个效果：

| 要素                 | 具体设计              | 作用                                                   |
| -------------------- | --------------------- | ------------------------------------------------------ |
| **积蓄计数器** | 10 次点燃触发一次爆炸 | 不是随机触发，而是"努力攻击 → 必然回报"，给玩家确定感 |
| **范围爆炸**   | 200% ATK 火焰 AoE     | 清小怪利器，对成群的骷髅/鼠群有巨大价值                |
| **反治疗**     | 命中目标治疗 -70%     | 核心克制机制：诅咒骑士的吸血和侍僧的治疗被压制         |
| **燃烧增伤**   | 对燃烧目标额外 +30%   | 2件和4件的联动：先点燃再爆炸，伤害最大化               |
| **冷却 6 秒**  | 防止连续爆炸过于无脑  | 爆炸间隔给敌人恢复窗口，保持紧张感                     |

 **战斗循环** ：普攻 → 触发点燃 → 积蓄计数 → 10次 → 爆炸清场 → 重新积蓄

---

### 套装二：「怨魂之怒」— 防御反击型（受伤越多越强）

 **核心幻想** ：每次被打都在积蓄怨念，怨念满溢时化为毁灭性的反击风暴，越危险越强大。

**2 件效果：怨念积蓄**

> 奥术抗性 +30%。每次受到伤害时获得 1 层「怨念」，每层增加  **3% 攻击力** ，最多叠  **8 层（24% ATK）** 。6 秒内未受到伤害则层数逐渐衰减（每秒减 1 层）。

* 不是纯防御，而是**受伤 → 变强**的转化机制
* 30% 奥术抗性保证不会被打死（第四章大部分怪是奥术元素）
* 8 层上限 = +24% ATK，比第三章烈焰之心的 +12% 翻倍，但需要持续受击维持
* 创造了有趣的心理：玩家从"不想被打"变成"被打也不亏"
* 6 秒衰减机制防止无限堆叠，保持动态博弈

**4 件效果：怨灵风暴 + 不灭之魂**

> ① 怨念叠满 **8 层**时，自动释放「怨灵风暴」：对周围所有敌人造成  **250% ATK 的奥术爆炸伤害** ，同时 **恢复自身 15% 最大 HP** 。触发后清空所有层数，内置冷却 10 秒。
>
> ② 受到致命伤害时 **免死一次** （每关重置）。触发免死后 **立即获得满层怨念** ，并在 3 秒内受到的伤害降低 50%。

逐层拆解：

| 要素               | 具体设计                      | 作用                                                                      |
| ------------------ | ----------------------------- | ------------------------------------------------------------------------- |
| **满层爆发** | 8 层 → 250% ATK 奥术 AoE     | 被打够了就反击，伤害比净魂爆燃更高（因为触发更难）                        |
| **爆发回血** | 恢复 15% 最大 HP              | 攻防一体：反击的同时续命，形成"扛伤→爆发→回血→再扛"循环                |
| **层数清空** | 爆发后从 0 重新积蓄           | 创造循环节奏，不是无脑常驻增伤                                            |
| **免死保底** | 致命伤免死一次/每关           | 继承第三章熔炉铸造的设计，是本章的生存底线                                |
| **免死联动** | 免死后立刻满层 + 3 秒减伤 50% | **高光时刻** ：濒死 → 免死 → 满层 → 立刻爆发反击。这是最燃的瞬间 |

 **战斗循环** ：被打 → 叠怨念 → 满8层 → 爆炸+回血 → 重新叠 → 循环。濒死时触发免死 → 立刻满层大爆发 → 翻盘。

## 对应关卡的套装需求曲线

| 关卡             | 推荐配置                       | 为什么                           |
| ---------------- | ------------------------------ | -------------------------------- |
| 1-3              | 第三章旧套装                   | 怪物简单，旧装够用               |
| 4（第一面墙）    | 净魂 2 件 或 怨魂 2 件         | 2 件效果的火伤/奥抗+增攻是质变   |
| 5 Boss 骨冠领主  | 净魂 2 件优先                  | Boss 火弱点-20%，灼烧 DoT 收益高 |
| 6-7              | 2+2 混搭 或 开始凑 4 件        | 侍僧+骑士组合需要反治疗或高爆发  |
| 8-9（终极墙）    | 任一套 4 件                    | 4 件效果是质变：反治疗或满层爆发 |
| 10 Boss 墓域君王 | **两套都可行，策略不同** | 见下方分析                       |

### 终极 Boss 的两种攻略路线

 **路线 A — 净魂烈焰 4 件** ：

* Boss 有轻微火抗(10%)，但灼烧+爆炸的持续伤害依然可观
* 反治疗 -70% 是克制 Boss「亡灵复生」（2%/s 回血）的关键
* 玩法：保持高攻速触发点燃 → 积蓄爆炸 → 压制回血
* 节奏更主动，需要输出够高

 **路线 B — 怨魂之怒 4 件** ：

* Boss 的「诅咒领域」降低玩家 25% 攻击力，但怨念层数能对冲
* 被 Boss 持续攻击 → 怨念快速叠满 → 250% ATK 爆发 + 回血续命
* 免死保底 → 满层立刻反击，给了一次翻盘机会
* 节奏更被动但更稳，适合装备数值不够但套装完整的玩家

这让终极 Boss 不是只有一种"标准答案"，两条路线都可以通关，体验不同。

## 实现层面的考量

需要新增的系统模块：

1. **DoT（持续伤害）系统** — 点燃效果需要给怪物挂一个定时伤害标记
2. **层数堆叠系统** — 怨念需要一个带计时器的层数管理器
3. **范围爆炸效果** — 两套 4 件都有 AoE 爆炸，可以复用同一个爆炸逻辑
4. **反治疗 debuff** — 给怪物加一个标记，治疗/吸血时检查乘数

其中 DoT 和层数堆叠是现有系统没有的，需要新建。但这两个系统是通用的——后续章节的套装也可以复用（比如第五章可以做毒 DoT 套装、冰冻层数套装等），投入产出比很高。
