## 设计理念

### 章节定位

| 维度 | 说明 |
|------|------|
| 主元素 | poison（毒） |
| 辅助元素 | physical（物理）、water（水） |
| 抗性系数 | 1.2（与第13章配对） |
| 色调 | {80, 180, 60}（暗毒绿） |

poison 作为主元素仅在第 7 章（瘴毒密林，抗性 0.85）出现过一次。第 14 章是 poison 的高阶回归，强度远超第 7 章，引入腐蚀叠层、回复压制、防御衰减等高级毒系机制。

### 与第 13 章（冰）的主题对比

| 维度 | 第13章 冰 | 第14章 毒 |
|------|----------|----------|
| 核心威胁 | 空间压缩 + 移速衰减 | 防御腐蚀 + DoT 叠加 |
| 压制方式 | 冻住你、困住你、让你跑不掉 | 毒蚀你、腐朽你、让你越来越脆 |
| 致死模式 | 被限制在小范围内吃满伤害 | DoT 和腐蚀叠加到阈值后雪崩式崩溃 |
| Build 偏好 | 高移速 + 冰抗 | 高韧性 + 毒抗 + 净化/回复 |
| 反制元素 | fire（融化反应） | fire（焚净反应） |

### 两个 Boss 的定位差异

| 维度 | 维诺莎（中 Boss） | 涅克洛斯（终 Boss） |
|------|-------------------|-------------------|
| 战斗风格 | 感染扩散型：密集弹幕叠毒 + 虫潮淹没 | 腐蚀碾压型：防御剥离 + DoT 雪崩 + 变异进化 |
| AI 决策重点 | 控制身上毒层数 + 清虫潮 + 打毒腺图腾 | 维持防御不被腐蚀清零 + 应对净化护盾 + 阶段三毒爆抢时 |
| Build 偏好 | 高攻速 + AOE → 快速清虫群减少毒叠加源 | 高韧性 + 毒抗 + 回复 → 抵抗腐蚀链 |
| 失败原因 | 毒层叠满 + 虫潮无法清理，DoT 致死 | 防御被腐蚀归零 + 毒爆引爆，一轮秒杀 |
| 阶段数 | 2 阶段 | 3 阶段 |

### 弱点

| 元素 | 抗性 | 说明 |
|------|------|------|

| poison | 0.50 | 极高抗 |

### 阶段一（100%→55%）：毒潮侵袭

Boss 以高频弹幕叠加毒层，同时召唤虫群形成数量压制。每颗弹命中都叠加"蚀毒"，蚀毒层数越高每秒 DoT 越痛，迫使玩家要么躲弹幕控制毒层、要么靠回复硬扛。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_barrage** | count: 18, spread: 140°, dmgMul: 0.70, speed: 190, interval: 5.5s |
| 地面威胁 | **ATK_spikes** | count: 4, radius: 30, delay: 1.0s, dmgMul: 1.0, lingerTime: 5s, interval: 7s |
| 召唤 | **SUM_minion** | monsterId: plague_mite, count: 6, interval: 8s |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| barrage.onHit | 每颗弹命中叠加 1 层"蚀毒"（每层 1.5% ATK/秒 DoT，持续 6s，最多 8 层）。新命中刷新全部层的持续时间 |
| spikes.linger | 残留毒池，站在上面每 0.5s 叠加 1 层蚀毒 + 减速 20%。毒池是持续的叠毒源 |
| minion.onDeath | 虫群死亡时释放毒雾（半径 25，持续 2s），范围内叠加 1 层蚀毒 |

**蚀毒机制详解**:
```
蚀毒 (Corrosive Venom):
- 每层: 1.5% ATK/秒 DoT
- 最大层数: 8 层 (= 12% ATK/秒 = 690×0.12 = 82.8 DPS)
- 持续时间: 6s (新命中刷新全部)
- 叠加源: 弹幕命中 / 毒池接触 / 虫群死亡毒雾
- 净化: 3s 不受新叠加，每秒自然消退 1 层
```

**AI 威胁信号**:
- 弹幕：每颗弹体注册 dangerZone（半径 10），AI 可横向移动躲避减少叠毒
- 毒池：预警阶段注册 dangerZone（半径 30，priority 0.8），比冰柱优先级更高——因为站在毒池里会持续叠毒
- 蚀毒层数：注册 debuffStack（currentStacks），AI 根据层数动态调整策略：低层(1-3)可激进，高层(5+)优先躲避

### 阶段二（55%→0%）：毒巢领域

弹幕从扇形变为环形（全方向），新增领域和毒腺图腾。图腾存活时强化全场蚀毒效果（每层 DoT +50%）。AI 面临三重决策：躲弹幕控毒层、脱离领域、打图腾削弱 DoT。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_barrage** | count: 14, spread: 360°（全向但更稀疏）, dmgMul: 0.70, speed: 160, interval: 6s |
| 领域 | **CTL_field** | radius: 110, dmgMul: 0.25, tickRate: 0.5s, duration: 7s, cd: 13s, hpThreshold: 0.55 |
| 减伤 | **DEF_armor** | hpThreshold: 0.40, dmgReduce: 0.60, duration: 5s, cd: 12s |
| 回血源 | **DEF_crystal** | count: 2, hpPct: 0.018, healPct: 0.012, spawnInterval: 11s, spawnRadius: 90 |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| field.effect | 领域内每 0.5s 叠加 1 层蚀毒（与弹幕独立叠加），同时回复压制 50%（领域内回血效果减半） |
| crystal.aura | 毒腺图腾存活时：全场蚀毒每层 DoT +50%（从 1.5% 变为 2.25%）。图腾 = DoT 倍率放大器 |
| crystal.onDestroy | 图腾摧毁时爆裂释放净化波，清除玩家 3 层蚀毒（打掉图腾有正面收益） |
| armor.visual | 毒液硬化甲壳，暗绿色护甲包裹效果 |

**AI 威胁信号**:
- 领域：dangerZone（半径 110，持续 7s）。领域内不仅掉血还叠毒+减回复，脱离优先级极高
- 毒腺图腾：priorityTarget（priority 0.75）。打掉图腾双重收益——中断 Boss 回血 + 削弱全场 DoT + 清除自身 3 层蚀毒
- 护甲期间：bossState（dmgReduce 0.60）。可趁此去清虫群/打图腾

**维诺莎战斗节奏**:
```
阶段一: 左右移动躲弹幕(140°扇形) + 走出毒池 + 清虫群(注意死亡毒雾)
        核心: 控制蚀毒层数在 3 层以下
    ↓ HP 55% 转换演出 (全场+2层蚀毒)
阶段二: 躲弹幕(360°更稀疏) + 脱离领域 + 优先打图腾(削弱DoT+清毒层)
        核心: 图腾存活时 DoT×1.5, 必须在毒层叠满前摧毁
```

### 阶段一（100%→60%）：腐蚀试探

Boss 以宽角度吐息和毒刺建立空间控制，同时召唤腐蚀守卫分散火力。吐息命中叠加"腐蚀"debuff（降低防御），与阶段二的 CTL_decay 形成连贯的防御剥离链。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_breath** | angle: 70°, range: 160, dmgMul: 0.45, tickRate: 0.3s, duration: 1.8s, interval: 7s |
| 地面威胁 | **ATK_spikes** | count: 3, radius: 35, delay: 1.2s, dmgMul: 1.1, lingerTime: 5s, interval: 9s |
| 嘲讽守卫 | **SUM_guard** | count: 2, hpPct: 0.012, atkMul: 0.35, tauntWeight: 0.55, interval: 14s |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| breath.onHit | 每次命中叠加"腐蚀"1 层（每层降低 DEF 4%，最多 10 层 = -40% DEF，持续 8s，新命中刷新）。为阶段二 CTL_decay 做铺垫 |
| spikes.linger | 残留毒沼，站在上面每秒受 0.3×ATK 毒伤 + 回复压制 40%（毒沼内回血效率大幅降低） |
| guard.aura | 守卫自带半径 45 的腐蚀光环（光环内每秒叠 1 层腐蚀），不清守卫则腐蚀快速叠满 |

**腐蚀机制详解**:
```
腐蚀 (Corrosion):
- 每层: DEF 降低 4%
- 最大层数: 10 层 (= DEF -40%)
- 持续时间: 8s (新命中刷新全部)
- 叠加源: 吐息命中 / 守卫光环 / 阶段二 decay 自动叠加
- 效果: 防御越低 → 受伤越多 → 与 DoT 形成恶性循环
```

**AI 威胁信号**:
- 吐息：dangerZone（扇形 70°，半径 160，持续 1.8s）。AI 可向侧面移动躲出，但吐息持续时间比冰章长 0.3s
- 毒沼：dangerZone（半径 35，priority 0.8）。毒沼的回复压制使得"硬站毒沼靠吸血扛"策略失效
- 守卫：taunt（weight 0.55）+ dangerZone（半径 45 光环）。守卫不仅嘲讽还自带腐蚀光环，必须优先清理

### 阶段二（60%→30%）：腐朽侵蚀

核心机制转变：CTL_decay 持续自动叠加腐蚀层，护盾考验元素构建，毒壁缩小安全区。与第 13 章阶段二对比——冰章压缩空间+减速，毒章剥离防御+压制回复。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_breath** | angle: 100°（扩大）, range: 160, dmgMul: 0.55, tickRate: 0.3s, duration: 1.8s, interval: 7s |
| 衰减 | **CTL_decay** | hpThreshold: 0.60, stat: def, reducePerSec: 0.015, maxReduce: 0.40, bonusOnHit: 0.04 |
| 障壁 | **CTL_barrier** | count: 2, duration: 7s, contactDmgMul: 0.35, interval: 13s |
| 领域 | **CTL_field** | radius: 130, dmgMul: 0.30, tickRate: 0.5s, duration: 9s, cd: 15s, hpThreshold: 0.60 |
| 反应护盾 | **DEF_shield** | hpPct: 0.035, bossDmgReduce: 0.80, duration: 10s, cd: 20s, hpThreshold: 0.50, baseResist: 0.50 |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| decay.stat | **def**（与冰章的 moveSpeed 不同——毒章削防御而非移速）。被吐息/毒沼命中时额外叠 4% |
| decay.combo | 腐蚀层与 decay 独立计算但效果叠加：decay -40% DEF + 腐蚀 10 层 -40% DEF = 理论 -80% DEF。实际受 DEF 下限保护（最低保留 10% 基础 DEF） |
| barrier.onContact | 碰触毒壁：中毒 2s（每秒 0.4×ATK 毒伤）+ 叠加 3 层腐蚀。毒壁比冰墙更致命 |
| field.effect | 领域内回复压制 60% + 每 0.5s 叠 1 层腐蚀。与 decay 叠加形成加速恶化 |
| shield.reaction | 见下方详细定义 |

**反应护盾特调（焚净反应）**:

```
shield_reaction: {
    weakReaction: "purify"           -- 火打毒 = 焚净
    weakElement: "fire"
    weakMultiplier: 2.5              -- 火元素对护盾 ×2.5 伤害

    wrongHitEffects: {
        poison: {                    -- 毒打毒盾：最重惩罚
            shieldHeal: 0.08         -- 护盾回复 8% 自身 HP
            bossHeal: 0.02           -- Boss 回复 2% maxHP
        }
        water: {                     -- 水打毒盾：毒素扩散
            spreadPoison: true       -- 扩散毒雾 AOE 半径 60
            aoeRadius: 60            -- 范围内叠 2 层腐蚀
        }
        ice: {                       -- 冰打毒盾：冰层覆盖但效率低
            dmgFactor: 0.6           -- 伤害 6 折
            slowSelf: 0.20           -- 自身减速 20%（3s）
        }
        physical: {                  -- 物理打毒盾：腐蚀反噬
            corrosion: 0.05          -- 每次命中玩家叠 5% DEF 降低
            maxStack: 5              -- 最多叠 5 次 = -25% DEF
        }
        arcane: {                    -- 奥术打毒盾：中性偏弱
            dmgFactor: 0.65          -- 伤害 6.5 折，无额外惩罚
        }
    }

    timeoutPenalty: {                -- 10s 内未击破
        type: "bossBuff"
        atkBonus: 0.30               -- Boss ATK +30%
        duration: 8.0                -- 持续 8s
    }
}
```

**AI 威胁信号**:
- 领域：dangerZone（半径 130）。领域内回复压制 + 叠腐蚀，必须脱离
- 毒壁：dangerZone（矩形）。碰触惩罚比冰墙更重（中毒 + 叠腐蚀），AI 将毒壁纳入移动禁区
- 护盾：priorityTarget（priority 0.8）+ bossState。AI 切换目标打护盾，火队优势
- 衰减：debuff。AI 感知 DEF 持续下降，可能切换为更保守的走位策略
- 腐蚀层数：debuffStack。AI 根据当前 DEF 剩余判断风险等级

### 阶段三（30%→0%）：腐朽终焉

全面碾压阶段。漩涡牵引把你拉进腐蚀核心，被 decay 叠满的防御降低使每一次受伤都是致命的。限时毒腺迫使 AI 在"清毒腺保命"和"抢 Boss 最后血量"间做抉择。与冰章阶段三对比——冰章用移速衰减困住你，毒章用防御衰减让你脆如纸。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 减伤 | **DEF_armor** | hpThreshold: 0.30, dmgReduce: 0.70, duration: 7s, cd: 15s |
| 回血 | **DEF_regen** | hpThreshold: 0.30, regenPct: 0.025 |
| 漩涡 | **CTL_vortex** | radius: 110, pullSpeed: 35, coreDmgMul: 0.55, coreRadius: 35, duration: 4.5s, interval: 11s |
| 限时引爆 | **ATK_detonate** | count: 3, hpPct: 0.01, timer: 9s, dmgMul: 2.2, bossHealPct: 0.08, interval: once |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| vortex.core | 核心区域每 0.3s 叠加 2 层腐蚀 + 回复压制 80%。防御已被 decay 削到低谷的玩家在核心区几乎必死 |
| detonate.onExplode | 爆炸后全场释放"腐蚀风暴"5s（每秒全场 0.5×ATK 毒伤 + 回复压制 100%）+ Boss 回复 8% HP |
| armor.visual | 腐蚀茧壳，暗绿色/紫色的厚重护甲包裹 |
| regen.note | 2.5%（低于冰章的 3%，因为毒章阶段三的 decay+腐蚀已经给足了拖延压力） |

**AI 威胁信号**:
- 漩涡：pull（radius 110, pullSpeed 35）。核心区的腐蚀叠加极快，AI 必须全力抵抗牵引
- 毒腺引爆：priorityTarget（priority 0.9）。不如冰章的 0.95（因为毒腺只有 3 个不是 4 个，难度略低），但"腐蚀风暴"惩罚极其恶毒——5 秒内完全无法回血
- 护甲+回血组合：decay 已削弱玩家防御，而 Boss 有减伤+回血，考验绝对 DPS 输出

**涅克洛斯战斗节奏**:
```
阶段一: 侧移躲吐息(70°) + 走出毒沼 + 优先清守卫(阻止光环叠腐蚀)
        核心: 控制腐蚀层数, 保持 DEF 健康
    ↓ HP 60% 转换演出 (全场+3层腐蚀)
阶段二: 在毒壁缩小的空间内躲吐息(100°) + 脱离领域 + 打焚净护盾(火队优势)
        同时 decay 持续削 DEF, 腐蚀层叠加越来越快
        核心: DEF 在雪崩前打掉护盾, 否则 Boss 增伤30%后更难扛
    ↓ HP 30% 转换演出 (全场+5层腐蚀)
阶段三: 抵抗漩涡牵引(避开核心区的疯狂叠腐蚀) + 9s内打掉3个毒腺
        毒腺爆炸 = 5s全场毒伤+完全禁止回血 + Boss回血
        护甲+回血 vs 玩家被削到极低DEF后的生存挣扎
```
