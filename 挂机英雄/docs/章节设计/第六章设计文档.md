## 第六章设计方案：「雷鸣荒漠」

### 主题定位

前五章分别是 荒原（灰黄）→ 冰封（冰蓝）→ 熔岩（熔红）→ 墓域（幽紫）→ 深海（深蓝），元素覆盖了火、冰、毒、水、物理、奥术。第六章走**雷暴沙漠**路线，理由：

* **琥珀金/电紫**色调与前五章形成鲜明视觉反差——温暖的沙漠金色配合电弧紫光，是全新的视觉组合
* **火+奥术**双元素主导：火在第三章后沉寂了两章，奥术在第四章后也退出了舞台，第六章让两者以"雷火"形态回归
* 沙漠雷暴生物（雷蝎、沙虫、电蜥、风暴鹰、沙漠傀儡）有丰富的视觉表现力
* 可引入新机制：**雷击链弹**（伤害在玩家召唤物/分身间传导）、**沙暴**（降低玩家命中/暴击）、**充能**（怪物受伤蓄能后释放强力反击）

```lua
    atkInterval = 1.2, element = "arcane",
    isRanged = true,

    image = "Textures/mobs/thunder_shaman.png",
    radius = 16, color = {200, 180, 255},

```

#### 新机制说明

| 机制 | 字段 | 作用 | 实现要点 |
|------|------|------|----------|
| **充能（chargeUp）** | `chargeUp = { stackMax, dmgMul, resetOnTrigger, isAOE?, aoeRadius? }` | 受击蓄能，满层后下次攻击强化 | 在 `OnEnemyTakeDamage` 中叠层；攻击时检查满层触发 |
| **链弹（chainLightning）** | `chainLightning = { bounces, dmgMul, element, range }` | 攻击后电弧弹射附近友方单位 | 在 `EnemyAttackPlayer` 后搜索玩家附近友方单位并造成伤害 |
| **沙暴（sandStorm）** | `sandStorm = { critReducePct, duration }` | 命中降低玩家暴击率 | 在 `EnemyAttackPlayer` 中施加 debuff，通过韧性衰减 |

**韧性交互**：所有新 debuff 都需要经过韧性抗性计算：
```lua
local resist = GameState.GetDebuffResist()
local actualCritReduce = critReducePct * (1 - resist)
local actualDuration = duration * (1 - resist * 0.5)
```

### 2 个 Boss 设计

#### 沙暴君主·拉赫（第 5 关 中章 Boss）

**设计理念**：沙暴中的统治者，大范围削弱 + 弹幕压制。2 件套是分水岭——没有套装抗性加成，沙暴 debuff 几乎无法承受。

```lua
boss_sandstorm_lord = {
    name = "沙暴君主·拉赫",
    isBoss = true,
    hp = 300000, atk = 130, def = 24, speed = 22,
    atkInterval = 1.5, element = "arcane",
    antiHeal = true,
    slowOnHit = 0.35, slowDuration = 2.0,

    -- 技能1: 雷暴弹幕 — 每7秒释放12颗电弧弹
    barrage = {
        interval = 7.0, count = 12, dmgMul = 0.5, element = "arcane",
    },

    -- 技能2: 沙暴领域 — HP<60% 时全场沙暴，降暴击率30%+降攻速15%
    frozenField = {
        hpThreshold = 0.6, slowRate = 0.40, duration = 6.0, cd = 16.0,
    },

    -- 技能3: 雷甲 — HP<40% 时获得减伤45%护甲
    iceArmor = {
        hpThreshold = 0.4, dmgReduce = 0.45, duration = 5.0, cd = 14.0,
    },

    -- 技能4: 召唤雷蝎
    summon = {
        interval = 10.0, monsterId = "thunder_scorpion", count = 3,
    },

    expDrop = 15000, goldDrop = {350, 500},
    image = "Textures/mobs/boss_sandstorm_lord.png",
    radius = 22, color = {200, 170, 60},

    resist = {
        fire = 0.20, ice = -0.20, poison = 0.10,
        water = -0.15, arcane = 0.50, physical = 0.25,
    },
}
```

**拉赫的核心挑战**：
1. 弹幕密度高（12颗/7秒），走位要求高
2. HP<60% 后领域减速+减暴击，暴击流受限
3. 召唤雷蝎会充能反击，分散注意力
4. **克制策略**：冰元素(-20%)或水元素(-15%)输出；潮汐壁垒2件的水抗回血在此无效，需要新套装

#### 雷霆泰坦·奥西曼（第 10 关 章末 Boss）

**设计理念**：远古雷电巨人，古代文明的战争兵器。兼具输出与生存，是整个第六章的终极检定。

```lua
boss_thunder_titan = {
    name = "雷霆泰坦·奥西曼",
    isBoss = true,
    hp = 650000, atk = 170, def = 40, speed = 16,
    atkInterval = 2.0, element = "fire",
    antiHeal = true,
    slowOnHit = 0.30, slowDuration = 2.5,

    -- 技能1: 雷火吐息 — 每8秒巨型火柱+电弧，3.0倍ATK
    dragonBreath = {
        interval = 8.0, dmgMul = 3.0, element = "fire",
    },

    -- 技能2: 雷暴领域 — HP<55% 全场电磁风暴，减速55%
    frozenField = {
        hpThreshold = 0.55, slowRate = 0.55, duration = 8.0, cd = 18.0,
    },

    -- 技能3: 充能护盾 — HP<35% 减伤55%，并每秒反弹10%伤害
    iceArmor = {
        hpThreshold = 0.35, dmgReduce = 0.55, duration = 5.0, cd = 15.0,
    },

    -- 技能4: 雷电再生 — HP<20% 每秒回复2.0%HP
    iceRegen = {
        hpThreshold = 0.20, regenPct = 0.020,
    },

    expDrop = 22000, goldDrop = {500, 750},
    image = "Textures/mobs/boss_thunder_titan.png",
    radius = 24, color = {255, 200, 50},

    resist = {
        fire = 0.40, ice = 0.10, poison = -0.15,
        water = -0.20, arcane = 0.30, physical = 0.20,
    },
}
```

**奥西曼的核心挑战**：
1. 龙息 3.0 倍 ATK 是历代最高，一旦命中几乎秒杀
2. 火抗 40%+奥术抗 30%，本章主元素对 Boss 无效，**必须找弱点**
3. 弱点是水(-20%)和毒(-15%)——迫使玩家反直觉地使用非本章元素
4. HP<20% 回血 2.0%/秒 = 13000HP/秒，需要爆发或反治疗
5. **克制策略**：水/毒元素武器 + 反治疗（沧溟之怒4件的腐蚀扩散可降DEF后再输出）

**两个 Boss 弱点差异化**：

| Boss | 主弱点 | 次弱点 | 免疫向 |
|------|--------|--------|--------|
| 拉赫 | 冰(-20%) | 水(-15%) | 奥术(50%) |
| 奥西曼 | 水(-20%) | 毒(-15%) | 火(40%) |

玩家无法用同一套元素策略通关两个 Boss，必须调整配置。

| 怪物 | 火 | 冰 | 毒 | 水 | 奥术 | 物理 | 设计理由 |

| **BOSS 奥西曼** | 40% | 10% | -15% | -20% | 30% | 20% | 雷火巨人抗火，弱水/毒 |

#### 关键规律

1. **奥术元素受限** — 风暴鹰(50%)、雷蝎(40%)、BOSS 拉赫(50%)高奥术抗，本章奥术不好用
2. **水元素最优** — 甲虫(-25%)、雷蝎(-20%)、蠕虫(-20%)、傀儡(-25%)、巫师(-20%)弱水，中章 Boss 也弱水(-15%)
3. **冰元素第二优** — 甲虫(-20%)、雷蝎(-15%)、风暴鹰(-25%)、蜥蜴(-20%)、巫师(-15%)弱冰，拉赫也弱冰(-20%)
4. **火元素有限** — 仅蛛后(-20%)和蠕虫(-15%)弱火，蜥蜴(30%)和奥西曼(40%)高火抗
5. **物理被高抗** — 甲虫(30%)、蠕虫(40%)、傀儡(50%)物理坦克成群，物理流在本章极为吃力
6. **Boss 弱点分化** — 拉赫弱冰/水，奥西曼弱水/毒，水是唯一通用元素

**策略总结**：本章主推**水元素**和**冰元素**武器。上一章（深海渊域）的水系装备在本章反而大放异彩——这是刻意的设计，让玩家不会完全抛弃旧装。

### 套装设计

#### 套装一：「风暴裁决」— 攻击型（雷电连锁 + 处决流）

**核心幻想**：驾驭雷电之力，将敌人标记后一击裁决。对低血目标拥有毁灭性的处决能力。

**ID**: `storm_verdict`
**主题色**: `{220, 180, 50}` (雷电金)
**所属章节**: 6

**2件效果：雷电标记**
> 攻击力 +15%。攻击命中时 25% 概率对目标施加「雷电标记」：被标记的目标受到的伤害增加 12%，持续 4 秒（不可叠加，可刷新）。

```lua
bonuses = {
    [2] = {
        desc = "攻击力+15%。攻击25%几率标记目标,受伤+12%,4秒",
        statsMul = { atk = 0.15 },
        buff = {
            trigger = "onHit",
            chance = 0.25,
            markDmgBonus = 0.12,
            markDuration = 4.0,
        },
    },
}
```

* ATK +15% 是直接战力提升，过渡阶段就有感知
* 雷电标记增伤 12% 是乘算，与其他增伤叠乘后效果显著
* 25%概率+4秒持续 = 攻速快的玩家几乎可以常驻标记

**4件效果：雷霆裁决**
> 对「雷电标记」状态的目标造成伤害时，若目标 HP < 30%，触发「雷霆裁决」：造成 **目标当前HP 15%** 的奥术真伤（无视抗性和防御），并引发电弧传导——对标记目标周围 80 范围内最多 3 个敌人各造成 60% ATK 的奥术伤害。每个目标每 4 秒最多触发一次裁决。

```lua
[4] = {
    desc = "标记目标HP<30%时触发裁决:15%当前HP真伤+电弧链3体60%ATK,CD4秒/目标",
    buff = {
        trigger = "onHitMarked",
        hpThreshold = 0.30,
        executeDmgPct = 0.15,  -- 15%当前HP真伤
        chainCount = 3,
        chainDmgMul = 0.60,
        chainElement = "arcane",
        chainRange = 80,
        cd = 4.0,
    },
},
```

| 要素 | 具体设计 | 作用 |
|------|---------|------|
| **处决真伤** | 15% 当前 HP，无视抗性/DEF | 越肉的怪越疼，完美克制本章高防坦克 |
| **电弧链弹** | 60%ATK × 3体，奥术元素 | 清场辅助，补充 AOE 能力 |
| **HP 阈值** | <30% 才触发 | 防止全程无脑触发，需要先打掉大半血 |
| **CD 限制** | 每目标 4 秒一次 | 控制 DPS 上限 |

**与本章怪物的联动**：
- 沙漠傀儡(350HP, 32DEF)和沙丘蠕虫(250HP, 26DEF)极难击杀 → 裁决真伤无视防御，精准克制
- 雷蝎充能 → 电弧链弹快速清理多只雷蝎，减少充能压力
- 雷能巫师治疗光环 → 标记增伤帮助快速集火击杀

#### 套装二：「磐岩之盾」— 防御型（岩甲 + 反震 + 不灭）

**核心幻想**：以大地之力铸就坚不可摧的岩甲，敌人的攻击如同敲击岩壁——只会伤到自己。

**ID**: `rock_aegis`
**主题色**: `{160, 130, 80}` (岩石棕金)
**所属章节**: 6

**2件效果：磐岩护体**
> 防御力 +20%，韧性 +10。受到物理伤害时，额外获得 3% 最大 HP 的岩甲护盾（每 3 秒最多触发一次），持续 5 秒。

```lua
bonuses = {
    [2] = {
        desc = "防御力+20%,韧性+10。受物理伤害获3%HP岩甲护盾(CD3s,5秒)",
        statsMul = { def = 0.20 },
        stats = { ten = 10 },
        buff = {
            trigger = "onTakePhysicalDmg",
            shieldPct = 0.03,
            shieldDuration = 5.0,
            cd = 3.0,
        },
    },
}
```

* DEF +20% 直接提升生存能力
* 韧性 +10（+20%抗性）对抗本章大量 debuff（沙暴减暴击、充能爆发、链弹）
* 物理伤害触发护盾——本章大量物理怪（甲虫、蠕虫、傀儡），触发率极高
* 与上一章潮汐壁垒不同：潮汐是水抗+受水伤回血，磐岩是物抗+受物伤护盾

**4件效果：大地之怒 + 磐岩不灭**
> ① 受到攻击时，35% 概率反震 **50% 受到伤害**为物理伤害（对攻击者），同时减少攻击者 10% 攻速，持续 3 秒。
> ② 受到致命伤害时**免死一次**（每关重置），触发后获得 3 秒无敌 + 立即恢复 20% 最大 HP，并释放地震波击退所有敌人。

```lua
[4] = {
    desc = "35%反震50%伤害+降攻速10%;致命伤免死(每关1次)+3秒无敌+回20%HP+地震击退",
    buff = {
        trigger = "onTakeDmg",
        reflectChance = 0.35,
        reflectPct = 0.50,
        atkSpeedReduce = 0.10,
        atkSpeedReduceDur = 3.0,
        fatalProtect = true,
        fatalInvulDur = 3.0,
        fatalHealPct = 0.20,
        fatalKnockback = true,
        perStage = true,
    },
},
```

| 要素 | 具体设计 | 作用 |
|------|---------|------|
| **反震伤害** | 35%概率反弹50%伤害 | 被群殴时反杀小怪，对蜂群甲虫效果极佳 |
| **降攻速** | -10%攻速/3秒 | 减缓敌方攻击频率，与 DEF 形成双重减伤层 |
| **免死保底** | 致命伤免死1次/关 | 延续历代防御套传统 |
| **3秒无敌** | 免死后3秒无敌 | 比上一章（潮汐冲击波击退）更强的喘息空间 |
| **20%HP回复** | 免死后立即回血 | 比上一章（15%HP回复）更高 |

**与本章怪物的联动**：
- 沙漠甲虫蜂群 → 反震 50% 可以反杀低血甲虫，群殴变群灭
- 沙漠傀儡充能 AOE → 3秒无敌可以完美规避充能爆发
- 雷蝎充能 ×2.5 攻击 → 免死保底防一击秒杀
- 风暴鹰穿透 30%DEF → DEF+20% 补偿部分穿透损失
