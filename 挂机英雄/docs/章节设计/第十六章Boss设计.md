## 设计理念

### 章节定位

| 维度 | 说明 |
|------|------|
| 主元素 | water（水） |
| 辅助元素 | ice（冰）、arcane（奥术） |
| 抗性系数 | 1.2（与第16-18章新三章配对） |
| 色调 | {30, 90, 200}（深渊蓝） |

water 在第 15 章（火）中是玩家的最优反制元素——Boss 护盾以 water 为最大弱点（淬灭反应 ×2.8）。第 16 章将水从"玩家的武器"反转为"敌人的武器"，迫使长期依赖水系 Build 的玩家重新构建阵容，与 ch15 将火反转的设计逻辑形成连续的元素博弈链条。

### 元素反转设计

| 维度 | 第15章（火） | 第16章（水） |
|------|------------|------------|
| 玩家最优元素 | water（淬灭） | **ice（冻结）** |
| Boss 最大弱点 | water (-0.25~-0.30) | **ice (-0.25~-0.30)** |
| 护盾弱点反应 | quench（水打火 ×2.8） | **freeze（冰打水 ×2.8）** |
| 设计意图 | 火队失效，必须转水 | 水队失效，必须转冰 |

### 与第 13-15 章的四章对比（两个三章循环）

| 维度 | 第13章 冰 | 第14章 毒 | 第15章 火 | **第16章 水** |
|------|----------|----------|----------|------------|
| 核心威胁 | 空间压缩 + 移速衰减 | 防御腐蚀 + DoT 叠加 | DPS 剥夺 + 受伤增幅 | **暴击削减 + 浸蚀叠层** |
| 压制方式 | 冻住你、困住你 | 毒蚀你、腐朽你 | 灼烧你、耗竭你 | **浸透你、熄灭你** |
| 致死模式 | 被限制在小范围吃满伤害 | DoT 和腐蚀叠加后雪崩崩溃 | 攻速归零+受伤增幅 | **暴击归零+DPS崩盘后被潮汐吞没** |
| Build 偏好 | 高移速 + 冰抗 | 高韧性 + 毒抗 | 高爆发 + 火抗 + 水系 | **高暴击基数 + 水抗 + 冰系** |
| 反制元素 | fire（融化反应） | fire（焚净反应） | water（淬灭反应） | **ice（冻结反应）** |
| decay 目标 | moveSpeed（跑不动） | def（扛不住） | atkSpeed（打不动） | **crit（暴不了）** |
| 战斗节奏 | 走位挑战 | 生存挑战 | DPS 竞速挑战 | **爆发窗口挑战** |

### 两个 Boss 的定位差异

| 维度 | 塞壬（中 Boss） | 勒维坦（终 Boss） |
|------|----------------|------------------|
| 战斗风格 | 潮汐倾泻型：高频水弹叠浸蚀 + 蟹群数量压制 | 深渊碾压型：暴击剥夺 + 潮汐涨落 + 万潮终焉 |
| AI 决策重点 | 控制浸蚀层数 + 清蟹群减少叠加源 + 打潮核打断增益 | 维持暴击不被耗竭 + 应对冻结护盾 + 阶段三潮核抢时 |
| Build 偏好 | 高暴击基数 + AOE → 快速清蟹群减少浸蚀源 | 高爆发 + 冰系 + 水抗 → 在暴击衰减前击杀 |
| 失败原因 | 浸蚀叠满 + 蟹群全场涌潮，暴击归零 DPS 崩盘 | 暴击被削到极低 + 潮汐增幅使敌人越来越强 |
| 阶段数 | 2 阶段 | 3 阶段 |

### 阶段一（100%→50%）：潮涌洗礼

Boss 以高频水弹铺满战场水压，同时召唤潮汐蟹群形成数量压制。每颗水弹命中都叠加"浸蚀"，浸蚀层数越高暴击率越低，迫使玩家要么精准走位控制浸蚀层数、要么在暴击归零前尽快击杀。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_barrage** | count: 18, spread: 140°, dmgMul: 0.80, speed: 190, interval: 5.5s |
| 地面威胁 | **ATK_spikes** | count: 5, radius: 30, delay: 1.0s, dmgMul: 1.0, lingerTime: 7s, interval: 7s |
| 召唤 | **SUM_minion** | monsterId: tidal_crab, count: 5, interval: 9s |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| barrage.onHit | 每颗水弹命中叠加 1 层"浸蚀"（每层暴击率 -2.5% + 火抗 -4%，持续 6s，最多 8 层）。新命中刷新全部层的持续时间 |
| spikes.linger | 残留潮汐水池，站在上面每 0.5s 叠加 1 层浸蚀 + 0.2×ATK 水伤。水池是持续的浸蚀叠加源 |
| minion.onDeath | 蟹群死亡时水花爆裂（半径 18，持续 2s），范围内叠加 1 层浸蚀 |

**浸蚀机制详解**:
```
浸蚀 (Drench):
- 每层: 暴击率 -2.5% + 火系抗性 -4%
- 最大层数: 8 层 (= 暴击率 -20% + 火抗 -32%)
- 持续时间: 6s (新命中刷新全部)
- 叠加源: 水弹命中 / 潮汐水池接触 / 蟹群死亡爆裂
- 净化: 4s 不受新叠加，每秒自然消退 1 层
- 与前三章的区别:
  ch13 frostbite: 减移速（走位失灵）
  ch14 蚀毒: DoT（持续掉血）
  ch15 灼烧: 减攻速+DoT（输出降低+持续掉血）
  ch16 浸蚀: 减暴击率（爆发能力崩盘，后期暴击是核心乘区）
```

**AI 威胁信号**:
- 水弹：每颗弹体注册 dangerZone（半径 10），AI 可横向移动躲避减少浸蚀叠加
- 水池：预警阶段注册 dangerZone（半径 30，priority 0.85），优先级高——水池内持续叠浸蚀
- 浸蚀层数：注册 debuffStack（currentStacks），AI 根据层数动态调整：低层(1-3)可激进输出，高层(5+)优先走位躲避

### 阶段二（50%→0%）：涌潮领域

弹幕从扇形变为环形（全方向），新增领域和潮核图腾。潮核存活时增强 Boss 攻击力（ATK +20%），使水弹和浸蚀叠加更有压力。AI 面临三重决策：躲水弹控浸蚀层、脱离领域、打潮核削弱 Boss 输出。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_barrage** | count: 14, spread: 360°（全向但更稀疏）, dmgMul: 0.80, speed: 165, interval: 6.5s |
| 领域 | **CTL_field** | radius: 115, dmgMul: 0.28, tickRate: 0.5s, duration: 8s, cd: 14s, hpThreshold: 0.50 |
| 减伤 | **DEF_armor** | hpThreshold: 0.35, dmgReduce: 0.58, duration: 5s, cd: 12s |
| 回血源 | **DEF_crystal** | count: 2, hpPct: 0.02, healPct: 0.015, spawnInterval: 12s, spawnRadius: 85 |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| field.effect | 领域内每 0.5s 叠加 1 层浸蚀（与水弹独立叠加），同时回复压制 40%（领域内回血效果降低） |
| crystal.aura | 潮核存活时：Boss ATK +20%（水弹伤害基数同步提升）。潮核 = Boss 输出倍率放大器 |
| crystal.onDestroy | 潮核摧毁时释放冰冻波，清除玩家 3 层浸蚀（打掉潮核有正面收益） |
| armor.visual | 硬化珊瑚甲壳，深蓝色珊瑚护甲包裹效果 |

**AI 威胁信号**:
- 领域：dangerZone（半径 115，持续 8s）。领域内掉血+叠浸蚀+减回复，脱离优先级高
- 潮核：priorityTarget（priority 0.75）。打掉潮核三重收益——中断 Boss 回血 + 削弱 Boss ATK + 清除自身 3 层浸蚀
- 护甲期间：bossState（dmgReduce 0.58）。可趁此去清蟹群/打潮核

**塞壬战斗节奏**:
```
阶段一: 左右移动躲水弹(140°扇形) + 走出水池 + 清蟹群(注意死亡水花)
        核心: 控制浸蚀层数在 3 层以下，保持暴击率健康
    ↓ HP 50% 转换演出 (全场+2层浸蚀)
阶段二: 躲水弹(360°更稀疏) + 脱离领域 + 优先打潮核(削弱Boss ATK+清浸蚀层)
        核心: 潮核存活时 Boss ATK×1.2, 浸蚀叠加更快, 必须在暴击归零前摧毁
```

### 阶段一（100%→60%）：深渊试探

Boss 以宽角度潮汐吐息和深海涌泉建立空间控制，同时召唤深渊守卫分散火力。吐息命中叠加"潮蚀"debuff（增加受到的水伤害），与阶段二的 CTL_decay 形成"暴不了+被水打更痛"的双重压制链。

| 技能槽 | 模板 | 参数 |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| breath.onHit | 每次命中叠加"潮蚀"1 层（每层增加受到的水系伤害 3.5%，最多 10 层 = +35% 水伤增幅，持续 8s，新命中刷新）。为阶段二 CTL_decay 做铺垫 |
| spikes.linger | 残留深海涌泉水池，站在上面每秒受 0.30×ATK 水伤 + 回复压制 40%（水池内回血效率大幅降低） |
| guard.aura | 守卫自带半径 48 的潮汐光环（光环内每秒叠 1 层潮蚀），不清守卫则潮蚀快速叠满 |

**潮蚀机制详解**:
```
潮蚀 (Tidal Corrosion):
- 每层: 受到的水系伤害 +3.5%（仅增幅水系伤害源，包含普攻、水弹、水池、环境伤害）
- 最大层数: 10 层 (= 水伤 +35%)
- 持续时间: 8s (新命中刷新全部)
- 叠加源: 吐息命中 / 守卫光环 / 阶段二 decay 被命中时额外叠加
- 效果: 水伤增幅 → 水系Boss/怪物伤害全部被放大 → 与潮汐领域形成正反馈
- 与焚灼的区别: 焚灼增幅全伤害(乘法)，潮蚀仅增幅水系伤害(更窄但更深)
```

**AI 威胁信号**:
- 吐息：dangerZone（扇形 70°，半径 160，持续 2.0s）。AI 可向侧面移动躲出
- 涌泉水池：dangerZone（半径 32，priority 0.8）。水池的回复压制使得"硬站水池靠吸血扛"策略失效
- 守卫：taunt（weight 0.55）+ dangerZone（半径 48 光环）。守卫不仅嘲讽还自带潮蚀光环，必须优先清理

### 阶段二（60%→30%）：深渊碾压

核心机制转变：CTL_decay 持续自动削减暴击率，护盾考验元素构建（要求 ice 而非 fire/water），潮汐壁缩小安全区。与前三章阶段二对比——冰章压缩空间+减速，毒章剥离防御+压制回复，火章剥夺攻速+增幅受伤，水章剥夺暴击+增幅水伤。四章形成"跑不动→扛不住→打不动→暴不了"的完整递进。

| 技能槽 | 模板 | 参数 |
|--------|------|------|
| 主攻 | **ATK_breath** | angle: 100°（扩大）, range: 160, dmgMul: 0.55, tickRate: 0.3s, duration: 2.0s, interval: 7s |
| 衰减 | **CTL_decay** | hpThreshold: 0.60, stat: crit, reducePerSec: 0.015, maxReduce: 0.35, bonusOnHit: 0.035 |
| 障壁 | **CTL_barrier** | count: 2, duration: 7s, contactDmgMul: 0.40, interval: 12s |
| 领域 | **CTL_field** | radius: 130, dmgMul: 0.32, tickRate: 0.5s, duration: 9s, cd: 15s, hpThreshold: 0.60 |
| 反应护盾 | **DEF_shield** | hpPct: 0.04, bossDmgReduce: 0.80, duration: 10s, cd: 20s, hpThreshold: 0.50, baseResist: 0.50 |

**章节特调**:

| 特调项 | 内容 |
|--------|------|
| decay.stat | **crit**（与冰章 moveSpeed、毒章 def、火章 atkSpeed 不同——水章削暴击率）。被吐息/水池命中时额外叠 3.5% |
| decay.combo | 暴击率衰减与潮蚀独立计算但效果协同：decay -35% crit（暴击几乎消失 → DPS 失去核心乘区）+ 潮蚀 10 层 +35% 水伤增幅（所有水系伤害更痛）= 既打不出暴击又被水系伤害碾压。实际暴击率下限保护（最低保留 5% 基础暴击率） |
| barrier.onContact | 碰触潮壁：水压冲击 2.5s（每秒 0.40×ATK 水伤）+ 叠加 3 层潮蚀。潮壁的惩罚直接暴力 |
| field.effect | 领域内回复压制 50% + 每 0.5s 叠 1 层潮蚀。与 decay 叠加形成加速恶化 |
| shield.reaction | 见下方详细定义 |

**反应护盾特调（冻结反应）**:

```
shield_reaction: {
    weakReaction: "freeze"           -- 冰打水 = 冻结
    weakElement: "ice"
    weakMultiplier: 2.8              -- 冰元素对护盾 ×2.8 伤害

    wrongHitEffects: {
        water: {                     -- 水打水盾：最重惩罚
            shieldHeal: 0.10         -- 护盾回复 10% 自身 HP
            bossHeal: 0.03           -- Boss 回复 3% maxHP
        }
        fire: {                      -- 火打水盾：蒸汽灼伤
            dmgFactor: 0.6           -- 伤害 6 折
            selfBurn: 0.012          -- 每秒自伤 1.2% maxHP（3s，蒸汽灼伤）
        }
        poison: {                    -- 毒打水盾：毒素被稀释
            dmgFactor: 0.5           -- 伤害 5 折
            drenchSelf: 2            -- 自身叠 2 层浸蚀
        }
        physical: {                  -- 物理打水盾：水流缓冲反弹
            reflect: 0.20            -- 反弹 20% 伤害给玩家
            critReduce: 0.08         -- 暴击率 -8%（3s）
        }
        arcane: {                    -- 奥术打水盾：中性偏弱
            dmgFactor: 0.65          -- 伤害 6.5 折，无额外惩罚
        }
    }

    timeoutPenalty: {                -- 10s 内未击破
        type: "tsunami"
        dmgMul: 1.5                  -- 护盾爆发潮汐对全场造成 1.5×ATK 水伤
        drenchStacks: 5              -- 全场叠加 5 层浸蚀
    }
}
```

**AI 威胁信号**:
- 领域：dangerZone（半径 130）。领域内回复压制 + 叠潮蚀，必须脱离
- 潮壁：dangerZone（矩形）。碰触惩罚极其暴力（持续水伤 + 叠潮蚀），AI 将潮壁纳入移动禁区
- 护盾：priorityTarget（priority 0.8）+ bossState。AI 切换目标打护盾，**冰队优势**（不再是水队或火队优势）
- 衰减：debuff。AI 感知暴击率持续下降，判断爆发窗口剩余时间，可能触发激进输出策略
- 潮蚀层数：debuffStack。AI 根据当前水伤增幅判断风险等级

### 阶段三（30%→0%）：万潮终焉

全面碾压阶段。漩涡将你拉入深渊核心，被 decay 叠满的暴击衰减使你几乎无法打出暴击，而潮蚀增幅让每一点水伤都被放大。限时潮核迫使 AI 在"打潮核保命"和"抢 Boss 最后血量"间做抉择。与前三章阶段三对比——冰章用移速衰减困住你，毒章用防御衰减让你脆如纸，火章用攻速衰减让你无力还击，水章用暴击衰减让你丧失爆发力。
